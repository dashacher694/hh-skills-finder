# HH Skills Finder Bot

Telegram-бот для анализа востребованных навыков на основе актуальных вакансий с HeadHunter.

## 🎯 Возможности

- 🔍 Поиск топ-15 навыков по любой профессии
- � Анализ зарплатной вилки (средняя, медианная, диапазон)
- 💼 Статистика по требуемому опыту работы
- � Анализ актуальных вакансий в реальном времени
- ⚡ Асинхронная обработка запросов
- 💾 Сохранение состояния диалога в Redis
- 📝 Детальная статистика по каждому навыку

## 🚀 Быстрый старт

### Требования

- Python 3.10+
- Redis
- Telegram Bot Token (получить у [@BotFather](https://t.me/BotFather))

### Установка

1. **Клонируйте репозиторий:**
   ```bash
   git clone <your-repo-url>
   cd hh-skills-finder
   ```

2. **Создайте виртуальное окружение:**
   ```bash
   python -m venv .venv
   .venv\Scripts\activate  # Windows
   source .venv/bin/activate  # Linux/Mac
   ```

3. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

4. **Настройте переменные окружения:**
   
   Создайте файл `.env` на основе `.env.example`:
   ```bash
   cp .env.example .env
   ```
   
   Отредактируйте `.env` и укажите ваш токен бота:
   ```env
   TELEGRAM_BOT_TOKEN=your_actual_bot_token
   REDIS_HOST=localhost
   REDIS_PORT=6379
   LOG_LEVEL=INFO
   ```

5. **Запустите Redis:**
   ```bash
   # Через Docker
   docker run -d -p 6379:6379 redis:alpine
   
   # Или установите Redis локально
   ```

6. **Запустите бота:**
   ```bash
   python main.py
   ```

## 📖 Использование

1. Найдите бота в Telegram
2. Отправьте `/start` для приветствия
3. Используйте `/find` для начала поиска
4. Введите название профессии (например, "Python разработчик")
5. Укажите количество вакансий для анализа (1-100)
6. Получите полный анализ:
   - 💰 Зарплатная вилка (средняя, медианная, мин-макс)
   - 💼 Требуемый опыт работы (распределение по категориям)
   - 🎯 Топ-15 востребованных навыков с количеством упоминаний

## 🛠️ Технологии

- **[aiogram 3.x](https://docs.aiogram.dev/)** - асинхронный фреймворк для Telegram Bot API
- **[aiohttp](https://docs.aiohttp.org/)** - асинхронный HTTP-клиент
- **[Redis](https://redis.io/)** - хранилище для FSM состояний
- **[HeadHunter API](https://dev.hh.ru/)** - источник данных о вакансиях

## ⚙️ Конфигурация

Все настройки задаются через переменные окружения в файле `.env`:

| Переменная | Описание | Значение по умолчанию |
|-----------|----------|----------------------|
| `TELEGRAM_BOT_TOKEN` | Токен Telegram бота | - (обязательно) |
| `REDIS_HOST` | Хост Redis сервера | localhost |
| `REDIS_PORT` | Порт Redis сервера | 6379 |
| `LOG_LEVEL` | Уровень логирования | INFO |
| `HH_API_URL` | URL HeadHunter API | https://api.hh.ru |
| `HH_USER_AGENT` | User-Agent для запросов | SkillsFinderBot/1.0 |
